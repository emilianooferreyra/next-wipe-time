**NextWipeTime** is a web application that tracks and displays up-to-date information about wipes (season/progression resets) for the most popular online games.

## What is it?

NextWipeTime aggregates information from multiple competitive and live-service games to help you know when the next season, event, or wipe begins. Features include:

- **20+ supported games**: Rust, Escape from Tarkov, Fortnite, Apex Legends, League of Legends, Path of Exile, Valorant, and many more
- **Real-time information**: Automated scrapers that fetch data from official sources, wikis, Reddit, and forums
- **Event calendar**: Centralized view of all upcoming events and seasons
- **Countdown timers**: Live countdown to the next wipe/season
- **Game filters**: Quickly find information for the game you're interested in
